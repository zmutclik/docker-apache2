1. 前言

本手册记述了利用Linux操作环境下的TM/BA系列打印机的驱动程序， 在TM-T82进行打印时，可以指定的打印条件。

有关Linux操作环境下的TM/BA系列打印机的驱动程序的信息， 请参阅打印机驱动程序手册。


2. 概述

TM-T82使用TM通用型号的"TM/BA通用热敏打印机"。

・CUPS WebAdmin追加打印机的时候选择的型号名：
"EPSON TM BA Thermal (rastertotmt) (en,en,zh_CN)"

・对应的PPD文件：
tm-ba-thermal-rastertotmt.ppd

您可以指定TM-T82的以下打印选项。

6. 蜂鸣器和钱箱使用时的打印机的设定

当使用蜂鸣器和钱箱，设定打印机可根据使用的选项的组合需要。
参照下表，请设置写在同一列，其中的选项状态相匹配的值。

选项状态
外部选项蜂鸣器 不使用 不使用 不使用 使用
钱箱#1 不使用 使用 不使用 不使用
钱箱#2 不使用 不使用 使用 不使用
数值名称值来设置 (* = 不关心)
BuzzerSelect * 0 0 1

注）

- BuzzerSelect : 0=不使用外部选项蜂鸣器, 1=使用外部选项蜂鸣器
- 外部蜂鸣器和钱箱不能同时使用。

有关设置打印机时，请使用端口设置工具tmpcsconfig在终端的命令行。
在设置打印机时，应正确连接，并且端口应设置正确。
你需要root权限来设定打印机。

# tmpcsconfig set 端口ID 数值名称 值来设置

示例）

- 对于连接到端口ESDPRT001打印机，设定BuzzerSelect=1。 BuzzerSelect=1.

# tmpcsconfig set esdprt001 buzzerselect 1

上述的缩写形式如下所示。

# tmpcsconfig s 1 bu 1
